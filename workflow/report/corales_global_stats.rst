Estadísticas globales de corales
================================

Este producto resume la capa base de corales antes de su integración regional al flujo de trabajo.

El archivo ``_global_stats.csv`` contiene estadísticas auxiliares usadas para normalizar o contextualizar la variable de corales en las reglas regionales de ``feature_corales``.

Notas técnicas
--------------

* Se calcula una sola vez para todo el workflow.
* Se usa como insumo común para las regiones procesadas.
* La salida se guarda en ``results/features/corales/_global_stats.csv``.
