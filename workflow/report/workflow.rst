Workflow IIE: reporte de ejecución
==================================

**Generación de capas e índice ecológico costero**

Este reporte documenta la ejecución del flujo de trabajo geoespacial para integrar variables ambientales,
procesar capas regionales y generar mapas finales.

Productos principales
---------------------

* Tablas regionales en formato Parquet.
* Tabla maestra de entrenamiento.
* Entrada para red bayesiana.
* Mapas finales `eicoastal_region_1.tif` a `eicoastal_region_14.tif`.
* Tabla `master_features_with_ie.parquet`.
* Reportes analíticos de las variables `iie-training-profile-py.html` e `iie-training-profile-R`.html
